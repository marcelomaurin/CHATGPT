## Обзор проекта

**TCHATGPT** — это open source набор визуальных и невизуальных компонентов для **Lazarus / Free Pascal**, созданный для упрощения интеграции возможностей искусственного интеллекта в настольные, промышленные, образовательные и корпоративные приложения.

Проект предоставляет компоненты для подключения к LLM-провайдерам, локальным моделям, обработки данных, машинного обучения, синтеза речи, обработки изображений, интеллектуальных агентов, графов, каналов ввода и вывода, а также экспериментальные компоненты для компьютерного зрения и 3D-графики.

> Этот проект следует понимать как **набор компонентов для интеграции ИИ в приложения Lazarus**, а не как полноценную AI-платформу, предназначенную для замены специализированных фреймворков обучения, MLOps-платформ или инфраструктуры масштабного развертывания моделей.

---

## Цель проекта

Основная цель проекта — дать разработчикам Lazarus / Free Pascal возможность добавлять функции искусственного интеллекта в свои системы простым, повторно используемым и компонентным способом.

Набор компонентов ориентирован на такие сценарии, как:

* создание ассистентов на базе генеративного ИИ;
* интеграция с LLM API;
* использование локальных моделей через совместимые серверы;
* генерация и анализ датасетов;
* простая классификация текстов;
* автоматизация с помощью интеллектуальных агентов;
* синтез речи;
* базовая обработка изображений;
* цифровые аудиофильтры;
* интеграция с устройствами, датчиками и внешними каналами;
* прототипирование AI-приложений в Lazarus.

## AI Core

Основные компоненты для генеративного ИИ, машинного обучения и поддержки проекта.

### `TCHATGPT`

Основной коннектор для провайдеров генеративного ИИ.

Позволяет отправлять prompts, настраивать провайдеров, выбирать модели и получать структурированные ответы.

Планируемые или поддерживаемые провайдеры:

* OpenAI;
* Google Gemini;
* Anthropic Claude;
* OpenRouter;
* Cerebras;
* локальный сервер, совместимый с `/v1/chat/completions`;
* Ollama или аналогичные локальные сервисы.

### `TNeuralNetwork`

Простая многослойная нейронная сеть, реализованная на Pascal.

Позволяет:

* создавать локальные нейронные сети;
* настраивать входы, скрытые слои и выходы;
* обучать по эпохам;
* вычислять loss;
* сохранять и загружать модели.

### `TTokenList`

Вспомогательный компонент для базовой токенизации текста.

Может использоваться для:

* классификации;
* анализа текста;
* предварительной обработки;
* графов принятия решений;
* подготовки датасетов.

### `TAICodeAssistant`

Ассистент кода на базе LLM.

Может использоваться для:

* проверки кода;
* предложения улучшений;
* генерации комментариев;
* объяснения блоков кода;
* помощи в тестировании;
* преобразования или документирования процедур.

### `TAIDatasetGenerator`

Генератор датасетов для обучения, fine-tuning или локальной классификации.

Поддерживает или планирует поддерживать структуры:

* CSV;
* JSON;
* JSONL;
* матрицы входов и выходов для локального обучения.

### `TAIModelRegistry`

Центральный реестр моделей, провайдеров, endpoints и параметров.

Помогает организовать:

* имя модели;
* провайдера;
* endpoint;
* temperature;
* лимит токенов;
* параметры по умолчанию.

### `TAIWizardConfig`

Мастер настройки для новых AI-проектов.

Может использоваться для подготовки проектов типа:

* chatbot;
* классификатор;
* pipeline;
* agent;
* технический ассистент.

## AI Sound Filters

Компоненты для цифровой обработки сигналов и аудиофильтрации.

### `TLowPassFilter`

Низкочастотный IIR-фильтр первого порядка.

Используется для сглаживания быстрых изменений и уменьшения высокочастотного шума.

### `THighPassFilter`

Высокочастотный IIR-фильтр первого порядка.

Используется для удаления низкочастотных компонентов, offset или DC-шума.

### `TAverageFilter`

Фильтр скользящего среднего.

Используется для простого сглаживания сигналов.

### `TFDMMultiplexer`

Компонент частотного мультиплексирования.

Позволяет моделировать каналы в разных частотных диапазонах.

### `TTDMMultiplexer`

Компонент временного мультиплексирования.

Позволяет чередовать каналы по временным слотам.

### `TCDMMultiplexer`

Компонент CDM/CDMA.

Использует ортогональные коды для разделения сигналов.

### `TOFDMMultiplexer`

OFDM-компонент с использованием FFT/IFFT.

Полезен для обучения, исследований и моделирования в области телекоммуникаций.

## AI Image

Компоненты для базовой обработки изображений.

### `TGrayscaleFilter`

Преобразует изображения в оттенки серого.

### `TNegativeFilter`

Применяет инверсию цветов.

### `TBrightnessContrastFilter`

Настраивает яркость и контраст.

Применяет пороговую обработку для получения черно-белого изображения.

### `TBlurFilter`

Применяет сглаживание через свертку.

### `TSharpenFilter`

Повышает резкость изображения с помощью ядра свертки.

### `TSobelFilter`

Обнаруживает края с помощью оператора Sobel.

### `TErosionDilationFilter`

Выполняет морфологические операции эрозии и дилатации.

## Установка пакета в Lazarus

1. Откройте Lazarus.
2. Перейдите в **Package > Open Package File (.lpk)**.
3. Выберите файл `pacote/packages/openai_core.lpk`.
4. Нажмите **Compile**.
5. Затем нажмите **Use > Install**.
6. Lazarus попросит пересобрать IDE.
7. После перезапуска компоненты появятся в палитре компонентов.

## Требования

### Основная среда

* Lazarus 3.x или выше;
* совместимая версия Free Pascal;
* Windows или Linux;
* пакет `openai_core.lpk`;
* интернет-соединение для внешних провайдеров;
* локальный сервер, настроенный при использовании offline-моделей.

### Windows

Для HTTPS-соединений могут потребоваться DLL OpenSSL, совместимые с архитектурой приложения.

Проверьте папку `pacote/lib/`.

Рекомендуется копировать необходимые DLL в ту же папку, где находится финальный исполняемый файл.

### Linux

В зависимости от используемых компонентов могут потребоваться дополнительные пакеты, такие как:

* OpenSSL;
* eSpeak/eSpeak-NG;
* libpython;
* библиотеки камеры или аудио;
* библиотеки для компьютерного зрения.

Требования могут отличаться в зависимости от компонента.

---

## Для кого проект пока не предназначен?

На текущем этапе проект не заменяет:

* полноценные machine learning frameworks;
* MLOps-платформы;
* корпоративные training pipelines;
* профессиональные сервисы развертывания моделей;
* специализированные библиотеки вроде PyTorch, TensorFlow, scikit-learn или полного OpenCV;
* enterprise AI-инфраструктуру.

## Заключение

**TCHATGPT** — перспективный набор компонентов для переноса возможностей ИИ в экосистему Lazarus / Free Pascal.

Его основная ценность заключается в практическом мосте между традиционными приложениями и современными AI-ресурсами, позволяя desktop, промышленным, образовательным и корпоративным системам интегрировать LLM, голос, изображения, графы, автоматизацию и локальные модели компонентным способом.

Проект все еще развивается, но уже имеет важную основу, чтобы стать open source ориентиром для AI-компонентов в Lazarus.
