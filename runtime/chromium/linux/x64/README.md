# CEF Runtime - Linux 64 bits

Este diretório armazena o runtime Chromium/CEF para Linux 64 bits.

## Arquivos Esperados na pasta in/
Os arquivos oficiais do CEF devem ser descompactados dentro da subpasta in/ local.
Não adicione DLLs, pacotes .so, .pak ou arquivos pesados de runtime ao controle de versão Git.
