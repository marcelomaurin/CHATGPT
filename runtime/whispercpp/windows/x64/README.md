# Whisper.cpp Runtime - Windows x64

Este diretório armazena os binarios oficiais do `whisper.cpp` para Windows 64 bits.

## Conteudo esperado em `bin/`

- `whisper-cli.exe`
- `main.exe`
- `whisper.dll`
- `ggml.dll`
- `ggml-base.dll`
- demais DLLs e utilitarios do release oficial

## Observacao

Os modelos `.bin` ou `.ggml` devem ser baixados separadamente e apontados em `WhisperCppModel`.
