# TAIGraphMap Demo — Classificação Textual por Grafo Ponderado

Este projeto demonstra visualmente o funcionamento do componente `TAIGraphMap`.

## Recursos do Demo
- Configurações da tokenização e do grafo (Minúsculas, Remoção de Acentos, Stopwords, Busca em Profundidade e Normalização).
- Treinamento a partir de um conjunto editável de exemplos de texto -> categoria.
- Classificação textual em tempo real.
- Visualização do Ranking de Categorias e Explicação baseada em evidências de pesos no grafo.
- Salvar/Carregar o Grafo e Dados de Treino em arquivos JSON.
