# kinect_frame_capture_test

Test console che usa i componenti Kinect del pacchetto per aprire il sensore, catturare il primo frame colore e salvarlo su disco.

## Output

- Log: `capture_output\kinect_frame_capture_test.log`
- Immagine: `capture_output\captured_color_frame.bmp`

## Quando usarlo

Usare questo test quando il demo grafico non mostra immagine o quando serve validare la cattura senza interfaccia visiva.

## Requisiti

Windows, Kinect SDK/Runtime 1.8, `Kinect10.dll` e Kinect collegato.
