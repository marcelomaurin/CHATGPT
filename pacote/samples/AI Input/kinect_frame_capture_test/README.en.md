# kinect_frame_capture_test

Console test that uses the package Kinect components to open the sensor, capture the first color frame, and save it to disk.

## Output

- Log: `capture_output\kinect_frame_capture_test.log`
- Image: `capture_output\captured_color_frame.bmp`

## When to use

Use this test when the graphical demo does not show an image or when you need to validate capture without a visual interface.

## Requirements

Windows, Kinect SDK/Runtime 1.8, `Kinect10.dll`, and a connected Kinect sensor.
