# kinect_frame_capture_test

Prueba de consola que usa los componentes Kinect del paquete para abrir el sensor, capturar el primer frame en color y guardarlo en disco.

## Salida

- Log: `capture_output\kinect_frame_capture_test.log`
- Imagen: `capture_output\captured_color_frame.bmp`

## Cuando usar

Use esta prueba cuando el demo grafico no muestre imagen o cuando necesite validar la captura sin interfaz visual.

## Requisitos

Windows, Kinect SDK/Runtime 1.8, `Kinect10.dll` y un Kinect conectado.
