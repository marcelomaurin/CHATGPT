# kinect_frame_capture_test

Test console qui utilise les composants Kinect du paquet pour ouvrir le capteur, capturer la premiere image couleur et l'enregistrer sur disque.

## Sortie

- Log: `capture_output\kinect_frame_capture_test.log`
- Image: `capture_output\captured_color_frame.bmp`

## Quand l'utiliser

Utilisez ce test si le demo graphique n'affiche pas d'image ou pour valider la capture sans interface visuelle.

## Prerequis

Windows, Kinect SDK/Runtime 1.8, `Kinect10.dll` et un Kinect connecte.
