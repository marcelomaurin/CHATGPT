# kinect_frame_capture_test

Консольный тест, который использует компоненты Kinect из пакета, открывает датчик, получает первый цветной кадр и сохраняет его на диск.

## Вывод

- Лог: `capture_output\kinect_frame_capture_test.log`
- Изображение: `capture_output\captured_color_frame.bmp`

## Когда использовать

Используйте этот тест, если графическое демо не показывает изображение или нужно проверить захват без GUI.

## Требования

Windows, Kinect SDK/Runtime 1.8, `Kinect10.dll` и подключенный Kinect.
