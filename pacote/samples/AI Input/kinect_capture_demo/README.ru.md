# kinect_capture_demo

Графический демо-проект для проверки цветного видеопотока Kinect через `TAIKinectSensor`, `TAIKinectColorStream` и, при необходимости, `TAIKinectDepthStream`.

## Назначение

Демо открывает датчик Kinect, запускает цветной поток, показывает кадры на экране и пишет события во вкладку **Log**. Опция **Depth** дополнительно включает поток глубины.

## Требования

- Windows.
- Установлен Kinect for Windows SDK/Runtime 1.8.
- `Kinect10.dll` доступна в системе.
- Приложение собрано под ту же архитектуру, что и установленная DLL.
- Датчик Kinect подключен и не занят другим приложением.

## Использование

1. Запустите `kinect_capture_demo.exe`.
2. Оставьте `Device` равным `0` для backend SDK10.
3. Включайте **Depth** только для проверки глубины.
4. Нажмите **Conectar**.
5. Смотрите изображение во вкладке **Video**.
6. Проверяйте сообщения во вкладке **Log**.
7. Нажмите **Desconectar** перед закрытием.

## Логи

Демо пишет события в `memLog` и читает внутренний лог backend:

`%TEMP%\aikinect_sdk10_backend.log`

В нем есть инициализация SDK, открытие потока, захват кадров, `LockRect`, `ReleaseFrame` и `NuiShutdown`.

## Диагностика

- Если изображения нет, проверьте **Log**.
- Если `NuiInitialize` завершается ошибкой, убедитесь, что Kinect не занят другим приложением.
- Если нет `Kinect10.dll`, переустановите Runtime/SDK 1.8.
- Для проверки только цветной камеры оставьте **Depth** выключенным.
