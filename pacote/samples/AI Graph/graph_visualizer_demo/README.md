# Graph Visualizer Demo (aigraphvisualizer)

This sample project demonstrates the usage of component `aigraphvisualizer, aigraphmap` from the `openai_graph` package.

## Features Illustrated
- Exercises at least 3 component properties.
- Calls at least 2 methods.
- Runs the real graph export flow directly, writing the Mermaid output to `graph_visual.mmd`.
- Draws the graph on screen with a simple node-and-link visualization.

## How to Build & Run
1. Open this project folder in Lazarus.
2. Verify package `openai_graph` is available or referenced.
3. Build the project (`Ctrl+F9` or run `lazbuild.exe`).
4. Execute and click "Atualizar grafo" to refresh the dataset and redraw the graph.
