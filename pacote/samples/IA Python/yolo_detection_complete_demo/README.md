# Yolo Detection Complete Demo (yolodetect)

This sample project demonstrates the usage of component `yolodetect` from the `openai_core` package.

## Features Illustrated
- Exercises at least 3 component properties.
- Calls at least 2 methods.
- Real execution of YOLOv8 object detector utilizing a Python interpreter.

## How to Build & Run
1. Open this project folder in Lazarus.
2. Verify package `openai_core` is available or referenced.
3. Build the project (`Ctrl+F9` or run `lazbuild.exe`).
4. Execute and click "Execute Test".
