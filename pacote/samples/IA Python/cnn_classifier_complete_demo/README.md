# Cnn Classifier Complete Demo (cnnclassifier)

This sample project demonstrates the usage of component `cnnclassifier` from the `openai_python` package.

> [!WARNING]
> **Compatibilidade (Apenas 64-bits)**: Este projeto **não funciona em ambiente de 32-bits (x86)**. O backend TensorFlow utilizado para a classificação de imagens não possui suporte ou builds oficiais para a plataforma Windows de 32-bits. O projeto deve ser compilado e executado obrigatoriamente como um aplicativo de 64-bits (`x86_64-win64`).

## Features Illustrated
- Exercises at least 3 component properties.
- Calls at least 2 methods.
- Supports **Simulation Mode** (enabled by default) to allow immediate verification of components without hardware, brokers, API credentials, or local Python installations.

## How to Build & Run
1. Open this project folder in Lazarus.
2. Verify package `openai_python` is available or referenced.
3. Certifique-se de compilar visando o target de 64-bits (`x86_64-win64`).
4. Build the project (`Ctrl+F9` or run `lazbuild.exe`).
5. Execute and click "Execute Test".
