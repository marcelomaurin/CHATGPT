## 🎨 Samples Organization

The demonstration projects are divided into:
1. **🖥️ GUI Projects**: Interactive Lazarus visual applications to configure properties, view graphics, capture camera frames, and run MLP, YOLO, and Kohonen networks in real-time.
2. **💻 Console Projects**: Simple `.lpr` command-line scripts to debug APIs, test tokenizers, run batch classification, and process CSV/JSONL datasets.

Explore individual subdirectories to check out direct Lazarus project files (`.lpi`, `.lpr`, `.pas`) and view screenshots.
