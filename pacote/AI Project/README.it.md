# [IA] Documentazione della Scheda AI Project

> [!NOTE]
> Questa cartella contiene la suite di componenti Lazarus sotto la scheda **AI Project**.

## Coordinamento Avanzato di Progetti IA e Pipeline di Esecuzione.
Centralizza e automatizza il flusso di lavoro tra i vari moduli.

### Riferimento Dettagliato dei Componenti

| Componente | Descrizione | Proprietà Importanti | Metodi Principali | Ruolo dell'Agente di IA |
|---|---|---|---|---|
| **TAIProject** | Global AI project coordinator and manager. | `ProjectName, Description, ChatGPT, Agent, Pipeline, DefaultProvider, DefaultModel, Token, LocalURL, SafeMode, SimulationMode` | `Initialize, TestConnection, ExecuteText, Execute, LoadFromFile, SaveToFile` | Centralize security keys, load JSON project setups, and execute routines. |
| **TAIProjectLLMConfig** | Manages the LLM provider and model configuration of the project. | `Project, Provider, Model, Endpoint, Token, SaveToken, Temperature, MaxTokens` | `ApplyToProject, LoadFromProject, LoadFromChatGPT, ValidateConfig, TestConnection` | Ease the configuration of the AI provider (OpenAI, local/Ollama, etc.). |
| **TAIProjectStorage** | Manages persistence and local storage of project data. | `Project, StoragePath, AutoSave, BackupOnSave` | `SaveProject, LoadProject, CreateBackup, DeleteProject` | Write and read the complete state of project planning and tasks on disk in JSON. |
| **TAIProjectDescription** | Stores the basic descriptive scope information of the project. | `Project, ProjectName, Goal, Context, Scope, Constraints, ExpectedDeliverables` | `LoadFromProject, SaveToProject, Clear` | Retain the initial parameters inputted by the user for plan generation. |
| **TAIAgileDocuments** | Manages agile documentation artifacts like Specifications and User Stories. | `Project, AutoGenerateDoc` | `GenerateMarkdownSpecification, ExportToPDF, SaveDocToFile` | Create and export formatted Markdown/PDF reports of agile project specifications. |
| **TAIProjectTasks** | Manages the list of tasks and activities generated for the project. | `Project, TasksCount` | `AddTask, RemoveTask, GetTaskByID, UpdateTaskStatus, ClearTasks` | Store the task list generated by AI, allowing status and progress updates. |
| **TAIProjectSpecification** | Generates structured agile specification using LLM. | `Project, PromptTemplate` | `GenerateSpecification, GetSpecificationMarkdown` | Elaborate detailed scope of requirements, epics, and acceptance criteria via AI. |
| **TAIProjectDependencies** | Analyzes and maps dependencies between project tasks. | `Project` | `AnalyzeDependencies, FindCircularDependencies, TopoSortTasks` | Ensure that the task execution sequence is logical and loop-free. |
| **TAIProjectAgents** | Manages the list of agent profiles allocated to the project. | `Project, AgentsCount` | `AddAgent, RemoveAgent, GetAgentByID, AssignTask` | Allocate the correct AI agent personas to the appropriate tasks. |
| **TAITaskActions** | Executes actions in a task's lifecycle. | `Project` | `ExecuteAction, StartTask, FinishTask, BlockTask, RejectTask` | Control state transitions (Start, Pause, Finish) of project tasks. |
| **TAIProjectReports** | Report generator component for project progress. | `Project, ReportFormat` | `GenerateProgressReport, GenerateGanttData, ExportReport` | Compile metrics and completion status for management reports. |
| **TAIProjectRevisions** | Controls the revision history and planning adjustments. | `Project, RevisionCount` | `AddRevision, RestoreRevision, GetRevisionDetails` | Track the modification history suggested by the user and AI replies. |
| **TAIProjectTaskGrid** | Visual grid control for displaying and editing tasks. | `Project, TaskGridColor` | `RefreshGrid, LoadFromProject` | Present the interactive task grid in the Lazarus interface. |
| **TAIProjectGantt** | Graphical control for rendering Gantt schedules. | `Project, HeaderColor, TaskBarColor` | `DrawGantt, ExportToImage` | Draw schedule bars visualizing task start and end dates. |
| **TAIProjectTimeline** | Timeline control with milestones and events. | `Project, AxisColor, NodeColor` | `DrawTimeline, AddEvent` | Present chronological milestones and executed actions. |
| **TAIRiskMatrix** | Visual risk matrix control (likelihood vs impact). | `Project, CellSize` | `DrawMatrix, LoadRisksFromProject` | Render the quadratic project risk matrix. |
| **TAIProjectStatusPanel** | Visual panel for progress and quick KPIs. | `Project, ShowGauges` | `UpdatePanel` | Show completion gauges and completed task counts. |
| **TAIAgentManagerFrame** | Visual control frame to manage AI agents. | `Project` | `LoadAgents, SaveAgents` | Provide ready-made UI to register and edit agents. |
| **TAITaskActionPanel** | Visual panel to trigger commands on tasks. | `Project, ActiveTaskID` | `LoadTask, ExecuteAction` | Render interactive buttons to advance task states. |
| **TAIProjectReportViewer** | Integrated report viewer. | `Project, ReportHTML` | `LoadReport, PrintReport` | Render reports and project plans on screen in rich formatting. |

### [Code] Esempio di Codice Lazarus (TAIProject)

```pascal
var
  MyProject: TAIProject;
  MySpecification: TAIProjectSpecification;
begin
  MyProject := TAIProject.Create(Self);
  MySpecification := TAIProjectSpecification.Create(Self);
  try
    MyProject.ProjectName := 'Smart Factory AI';
    MyProject.ChatGPT := ChatGPT1;
    
    MySpecification.Project := MyProject;
    
    if MyProject.Execute then
      ShowMessage('Sucesso')
    else
      ShowMessage(MyProject.LastError);
  finally
    MySpecification.Free;
    MyProject.Free;
  end;
end;
```


### [Bridge] Ponte di IA e Pianificazione
Ciascuno di questi componentes include una proprietà published `Prompt` que documenta in modo trasparente le proprie API per orientare gli Agenti IA (`TAIAgent`) autonomamente.
