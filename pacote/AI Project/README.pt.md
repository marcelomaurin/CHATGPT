# [IA] Documentação da Aba AI Project

> [!NOTE]
> Esta pasta contém a paleta de componentes do Lazarus sob a aba **AI Project**.

## Coordenação Avançada de Projetos de IA e Pipelines de Execução.
Centraliza e automatiza a conexão entre os diversos módulos do projeto.

### Referência Detalhada dos Componentes

| Componente | Descrição | Propriedades Importantes | Métodos Principais | Papel do Agente de IA |
|---|---|---|---|---|
| **TAIProject** | Coordenador e cérebro global do projeto de IA. | `ProjectName, Description, ChatGPT, Agent, Pipeline, DefaultProvider, DefaultModel, Token, LocalURL, SafeMode, SimulationMode` | `Initialize, TestConnection, ExecuteText, Execute, LoadFromFile, SaveToFile` | Centralizar chaves de segurança, carregar configurações de arquivos JSON e disparar execuções. |
| **TAIProjectLLMConfig** | Gerencia a configuração do provedor e modelo de LLM do projeto. | `Project, Provider, Model, Endpoint, Token, SaveToken, Temperature, MaxTokens` | `ApplyToProject, LoadFromProject, LoadFromChatGPT, ValidateConfig, TestConnection` | Facilitar a parametrização do provedor de inteligência artificial (OpenAI, local/Ollama, etc.). |
| **TAIProjectStorage** | Gerencia a persistência e armazenamento local de dados do projeto. | `Project, StoragePath, AutoSave, BackupOnSave` | `SaveProject, LoadProject, CreateBackup, DeleteProject` | Gravar e ler o estado completo do planejamento e tarefas do projeto no disco em formato JSON. |
| **TAIProjectDescription** | Armazena as informações descritivas básicas do escopo do projeto. | `Project, ProjectName, Goal, Context, Scope, Constraints, ExpectedDeliverables` | `LoadFromProject, SaveToProject, Clear` | Guardar os parâmetros iniciais inseridos pelo usuário para a geração do plano. |
| **TAIAgileDocuments** | Gerencia artefatos de documentação ágil como Especificações e Histórias de Usuário. | `Project, AutoGenerateDoc` | `GenerateMarkdownSpecification, ExportToPDF, SaveDocToFile` | Criar e exportar relatórios formatados em Markdown/PDF das especificações ágeis do projeto. |
| **TAIProjectTasks** | Gerencia a lista de tarefas e atividades geradas para o projeto. | `Project, TasksCount` | `AddTask, RemoveTask, GetTaskByID, UpdateTaskStatus, ClearTasks` | Armazenar a lista de tarefas gerada pela IA, permitindo atualizar progresso e estados. |
| **TAIProjectSpecification** | Gera a especificação ágil estruturada usando LLM. | `Project, PromptTemplate` | `GenerateSpecification, GetSpecificationMarkdown` | Elaborar o escopo detalhado de requisitos, épicos e critérios de aceite via IA. |
| **TAIProjectDependencies** | Analisa e mapeia dependências entre as tarefas do projeto. | `Project` | `AnalyzeDependencies, FindCircularDependencies, TopoSortTasks` | Garantir que a ordem de execução das tarefas seja lógica e sequencial livre de loops. |
| **TAIProjectAgents** | Gerencia a lista de perfis de agentes alocados para o projeto. | `Project, AgentsCount` | `AddAgent, RemoveAgent, GetAgentByID, AssignTask` | Alocar as personas de agentes de IA corretas para as tarefas apropriadas. |
| **TAITaskActions** | Executa ações no ciclo de vida de uma tarefa. | `Project` | `ExecuteAction, StartTask, FinishTask, BlockTask, RejectTask` | Controlar as mudanças de estado (Iniciar, Pausar, Finalizar) das tarefas do projeto. |
| **TAIProjectReports** | Componente gerador de relatórios de progresso do projeto. | `Project, ReportFormat` | `GenerateProgressReport, GenerateGanttData, ExportReport` | Compilar métricas e status de conclusão para visualização de relatórios gerenciais. |
| **TAIProjectRevisions** | Controla o histórico de revisões e alterações no planejamento. | `Project, RevisionCount` | `AddRevision, RestoreRevision, GetRevisionDetails` | Rastrear o histórico de modificações sugeridas pelo usuário e respostas da IA. |
| **TAIProjectTaskGrid** | Controle visual para exibição e edição de tarefas em tabela. | `Project, TaskGridColor` | `RefreshGrid, LoadFromProject` | Apresentar a grade de tarefas interativa na interface do Lazarus. |
| **TAIProjectGantt** | Controle gráfico para exibição de cronograma Gantt. | `Project, HeaderColor, TaskBarColor` | `DrawGantt, ExportToImage` | Desenhar as barras de cronograma visualizando datas de início e fim das tarefas. |
| **TAIProjectTimeline** | Controle de linha do tempo com marcos e milestones. | `Project, AxisColor, NodeColor` | `DrawTimeline, AddEvent` | Apresentar marcos históricos e ações executadas cronologicamente. |
| **TAIRiskMatrix** | Controle visual de matriz de riscos (probabilidade x impacto). | `Project, CellSize` | `DrawMatrix, LoadRisksFromProject` | Renderizar o gráfico quadrático de riscos do projeto. |
| **TAIProjectStatusPanel** | Painel visual de progresso e KPIs rápidos. | `Project, ShowGauges` | `UpdatePanel` | Mostrar gauges de conclusão e quantidade de tarefas concluídas. |
| **TAIAgentManagerFrame** | Frame de controle visual para gerenciar agentes de IA. | `Project` | `LoadAgents, SaveAgents` | Disponibilizar interface pronta para cadastrar e editar agentes. |
| **TAITaskActionPanel** | Painel visual para acionar comandos sobre tarefas. | `Project, ActiveTaskID` | `LoadTask, ExecuteAction` | Renderizar botões interativos para avançar o estado das tarefas. |
| **TAIProjectReportViewer** | Visualizador integrado de relatórios. | `Project, ReportHTML` | `LoadReport, PrintReport` | Renderizar relatórios e planos de projeto na tela em formato rico. |

### [Code] Exemplo de Código Lazarus (TAIProject)

```pascal
var
  MyProject: TAIProject;
  MySpecification: TAIProjectSpecification;
begin
  MyProject := TAIProject.Create(Self);
  MySpecification := TAIProjectSpecification.Create(Self);
  try
    MyProject.ProjectName := 'Smart Factory AI';
    MyProject.ChatGPT := ChatGPT1;
    
    MySpecification.Project := MyProject;
    
    if MyProject.Execute then
      ShowMessage('Sucesso')
    else
      ShowMessage(MyProject.LastError);
  finally
    MySpecification.Free;
    MyProject.Free;
  end;
end;
```


### [Bridge] Ponte de IA e Planejamento
Cada um destes componentes possui uma propriedade published `Prompt` que documenta sua API de forma transparente para orientar Agentes de IA (`TAIAgent`) de forma automática!
