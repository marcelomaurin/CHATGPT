# Novidades - 07 de Julho de 2026

## Componente AIUSB (Beta)
- **Correção Crítica de Compatibilidade no Windows**: O enumerador de dispositivos USB foi reescrito para utilizar chamadas nativas à API de registro do Windows (`SYSTEM\CurrentControlSet\Enum\USB`) em vez de invocar o utilitário depreciado `wmic.exe`. 
- Isso resolve o problema onde o demo `aiusb_devices_demo` não listava nenhum dispositivo em instalações modernas do Windows 10 e 11.
- O demo foi atualizado e está classificado como **Beta**.

## Suporte a Kinect (Família AIKinect)
- **Integração Real com Driver Microsoft**: Adicionado suporte para capturar imagens e mapas de profundidade reais do Kinect v1 usando a DLL oficial da Microsoft (`Kinect10.dll`) no Windows.
- **Mascaramento de Exceções de FPU**: Envolvidas as chamadas de inicialização do sensor e abertura de streams em máscaras de exceção de ponto flutuante, evitando crashes de divisão por zero (`EZeroDivide`) no Lazarus/FPC.
- **COM Reference Counting**: Corrigido o gerenciamento de contagem de referências das texturas do Kinect para evitar liberação prematura do buffer de imagem no Free Pascal.
